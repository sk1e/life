# Game of life

## Installation

```sh
git clone https://github.com/sk1e/life && cd life && git checkout refactoring && npm i
    
 ```
 
## Build

### Production

```sh
npm run build
    
```


### Development

```sh
npm run build-dev
 ```

## Testing

```sh
npm run test
 ```


## Launching

```sh
npm run start
 ```
